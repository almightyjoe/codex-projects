# Legal Boundaries

This project is a clean-room reimplementation of a Windows network monitoring workflow.

## Allowed

- Use public documentation to understand product capabilities.
- Observe installed application behavior at the UI and data level.
- Inspect local user-owned settings, logs, and database shape to understand workflows.
- Build original code with independently selected libraries and services.

## Not Allowed

- Decompile, disassemble, patch, or modify Stardock binaries.
- Reuse Stardock assets, icons, proprietary data, branding, copy, license files, or service endpoints.
- Circumvent licensing, activation, updates, or telemetry controls.
- Redistribute the Stardock installer or installed files.

## Naming

The working project name is `Connection Observer` to avoid presenting the app as an official Stardock product.
